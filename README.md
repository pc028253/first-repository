# 我的第一個 GitHub 倉庫

這是我的第一個 GitHub 倉庫，用於學習和實踐 Git 的基本操作。

## 項目簡介

這個倉庫將用於：
- 學習 Git 基本命令
- 練習版本控制
- 熟悉 GitHub 操作

## 如何開始

1. 克隆倉庫
```bash
git clone https://github.com/pc028253/first-repository.git
```

2. 進入項目目錄
```bash
cd first-repository
```
